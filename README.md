# Transportation Data Analysis & IDM Simulation

## Overview
This project analyzes transportation data using NHTS and NGSIM datasets. 
It includes data cleaning, visualization, and a traffic simulation.

## Files
- Warsame_project3.ipynb (main code)
- NHTS(in).csv (dataset)
- NGSIM(in).csv (dataset)
- Transportation_Report.docx
- Scope_of_Work.docx
- annotated_code.docx

## How to Run
Open the notebook and run all cells.
